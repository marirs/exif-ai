Command Line

Powerful CLI for automation

Process images from the terminal. Supports dry run, JSON output, batch processing, and more.

Terminal 
# Generate a default config file
exif-ai-cli --init

# Preview what AI would write (dry run)
exif-ai-cli --dry-run photo.jpg

# Process for real
exif-ai-cli photo.jpg

# Process an entire folder
exif-ai-cli ./photos/

# JSON output for scripting
exif-ai-cli --json photo1.jpg photo2.jpg

# Use local AI â€” no API keys needed
exif-ai-cli --download-model
exif-ai-cli photo.jpg
